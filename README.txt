Procedure:

1. to create a convolved C2H2 model gird, run createconv.py

2. to change the location of the C2H2 slab, change the "loca" variable in all python scripts in python/2012 to your desire, default is 10 (Rin)

3. to combine GRAMS and C2H2, run all python scripts in python/2012 to create 10 fits files in fits/2012

4. run all .sh files in mask/2012 to do the fitting procedure (takes around a day)

5. to draw graphs, see the docstring in each python script in python/
